Stack Rank
==============

This a daily stack rank.

Requirements
------------

-	[Vagrant](https://www.vagrantup.com/docs/installation/)

Usage
-----

### Start Project

```bash
$> git clone git@github.com:MuleDojo/stackrank.git
$> cd stackrank
$> vagrant up
```

The open your browser http://127.0.0.1:8080

### Shutdown Project

```bash
$> vagrant halt
```
